## Player.ini说明
```ini
[Init]
Name=南宫飞云（姓名，必须是南宫飞云）
Kind=2（类别，主角是2）
NpcIni=z-南宫飞云.ini
Dir=0（方向，0～7，一共8种）
MapX=76（横坐标）
MapY=184（纵坐标）
Action=0
WalkSpeed=1（行走速度）
PathFinder=1
DialogRadius=1
ScriptFile=
State=0
Relation=0
Life=4000（生命值）
LifeMax=4000（生命满值）
Thew=2000（体力值）
ThewMax=2000（体力满值）
Mana=3000（内力值）
ManaMax=3000（内力满值）
Attack=4500（攻击力）
Defend=3500（防御力）
Evade=1000（身法）
Exp=0（已获得经验）
LevelUpExp=0（升级所需经验）
Level=40（当前人物等级）
AttackLevel=100（普通攻击武功等级）
Lum=0（人物自身颜色）
VisionRadius=10
AttackRadius=1
BodyIni=z01-南宫飞云尸体.ini
FlyIni=player-magic-长剑.ini
DeathScript=主角死亡.txt（死亡脚本）
Money=9000（金钱数）
LevelIni=level-easy.ini（难度代码）
```
